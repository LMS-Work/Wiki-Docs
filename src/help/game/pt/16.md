# 饰品
### 主要饰品由三个MOD提供“神秘遗物”，“奇异饰品”，“遗物”
# 饰品位置
## 按E打开背包界面，点击人物图像左上方白点即可打开饰品栏（已增加普通饰品栏数量，可同时佩戴普通不重复的饰品，同类饰品效果不叠加）
![饰品栏](https://i1.mcobj.com/imgb/u15prb/20240715_669520d39ca8c.png)
# 神秘遗物
## 在玩家第一次进入世界时将会获得神秘护符和七咒之戒

![神秘护符](https://i1.mcobj.com/imgb/u15prb/20240714_6693e1eeb3e42.png)

![七咒之戒](https://i1.mcobj.com/imgb/u15prb/20240714_6693e30e6b4aa.png)
## 七罪之戒对游戏影响很大，是否使用请慎重考虑，详细内容通过合成启示之证了解，一些合成物品只可以在启示之证中了解
![启示之证](https://i1.mcobj.com/imgb/u15prb/20240715_6694d5517e76b.png)
### 注：术石和卷轴均只有一个槽位，不可重复佩戴，且由于MOD冲突，传送世界时部分卷轴饰品（如千咒卷轴，附魔师的珍珠）会自动掉落入背包，请确保背包内留有空间。
# 奇异饰品
## 此MOD里的饰品在所有世界的战利品箱里均有概论获得，且获得时可看到详细功能，也可在JEI中查看
![奇异饰品](https://i1.mcobj.com/imgb/u15prb/20240715_669523bdb9814.png)
### 注：在主世界地下洞穴里会刷新宝箱怪，击杀时必定会掉落一件该MOD随机饰品
# 遗物
## 改MOD内饰品只可在研究台（研究和升级模组内饰品、物品的功能方块）上查看功能，在手上右键放在研究台上，下蹲加右键查功能和升级。
![研究台](https://i1.mcobj.com/imgb/u15prb/20240715_669525ad27d6a.png)
![研究饰品](https://i1.mcobj.com/imgb/u15prb/20240715_66952605c9a8f.png)
### 可用玩家经验给饰品升级，饰品有星级区别，用经验刷新，给饰品升级也需要经验
